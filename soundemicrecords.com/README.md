Soundemic Records — Web Application

A clean and lightweight static website designed for showcasing music, artists, events, or any type of creative content. This project is simple, fast, and fully front-end based — making it easy to host on any server or even on GitHub Pages.

    Features

Fully static website built with HTML, CSS, and JavaScript

Responsive design so it works on desktop and mobile

Clean structure, easy to understand and extend

Custom JavaScript logic for interactive UI behavior

Modern layout suitable for a music label, creative studio, or portfolio

    Project Structure soundemicrecords.com/ │ ├── index.html # Main webpage ├── css/ │ └── style.css # Stylesheet (layout, colors, responsiveness) └── js/ └── script.js # Front-end interactivity and UI logic

    How to Run the Project

This website requires no backend and no special installation.

✔️ Option 1: Open directly

Just double-click index.html — it will open in your browser.

✔️ Option 2: Serve locally (recommended)

If you want proper local hosting:

Using Python
For Python 3

python3 -m http.server 8080

Then open:

http://localhost:8080

Using Node.js live server (if installed) npx live-server

    Deployment
